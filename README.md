# Z80 Playground

This is merely a playground for Z80 assembly. For the time being, [`pasmo`][1] is
used to assemble the sources and [`fuse`][2] is used to run them.

[1]: http://pasmo.speccy.org/
[2]: http://fuse-emulator.sourceforge.net/
